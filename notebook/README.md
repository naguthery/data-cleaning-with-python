# Student Success Data Cleaning Notebook

This folder contains the Google Colab notebook used for the **Student Success Data Cleaning** project. The notebook demonstrates the complete data cleaning workflow performed in Python to improve data quality and prepare the dataset for future analysis.

## Overview

The notebook performs several data cleaning tasks, including:

* Importing the raw dataset
* Inspecting the data structure and data types
* Identifying and handling missing values
* Removing duplicate records
* Standardizing categorical values
* Correcting formatting inconsistencies
* Validating the cleaned dataset
* Exporting the final cleaned dataset as a CSV file

## Tools and Libraries

* Python
* Google Colab
* pandas
* NumPy

## Files

* 'University_Cleaning.ipynb' – Google Colab notebook containing the complete data cleaning workflow.

## Outcome

The notebook transforms the raw dataset into a clean, consistent, and analysis-ready dataset that can be used for future analytics, visualization, or machine learning projects.
