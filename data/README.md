# Student Success Data

This folder contains the datasets used in the **Student Success Data Cleaning** project. The purpose of this project was to clean and prepare a raw student success dataset using Python by identifying and correcting data quality issues, standardizing values, and creating a reliable dataset for future analysis.

## Contents

* `student_success_messy.csv` – Original raw dataset containing missing values, inconsistencies, and formatting issues prior to cleaning.
* `student_success_clean.csv` – Final cleaned dataset produced after the data cleaning process and ready for future analysis.
