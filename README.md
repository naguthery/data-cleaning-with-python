## Project Overview

The **Student Success Data Cleaning** project demonstrates a complete data cleaning workflow using Python. The objective was to transform a raw student success dataset into a clean, consistent, and reliable dataset by identifying and correcting common data quality issues. The resulting dataset is suitable for future data analysis, visualization, and machine learning applications.

## Project Objectives

* Assess the quality of the original dataset.
* Identify missing values, duplicate records, and inconsistent data.
* Standardize formatting and categorical values.
* Produce a clean, analysis-ready dataset.
* Document the data cleaning process and methodology.

## Data Cleaning Process

The project followed a structured data cleaning workflow that included:

1. Importing and exploring the dataset.
2. Inspecting data types and overall data quality.
3. Identifying and handling missing values.
4. Removing duplicate records.
5. Standardizing categorical values and formatting.
6. Validating the cleaned dataset.
7. Exporting the cleaned dataset for future use.
